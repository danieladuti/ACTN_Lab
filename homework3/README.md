## Homework requirement
1. Implement the Solovay-Strassen primality test. For computing the Jacobi symbol, use the rules discussed in class (5p)

2. Implement the Lucas-Lehmer primality test for Mersenne numbers. For performing the modular reduction modulo a Mersenne number, use the algorithm discussed in class  (5p)


## Grading scale:
### Solovay-Strassen primality test
- own implementation of the Jacobi symbol calculation using the rules discussed in the course and testing this component (3p)
- the actual test (2p)

### Lucas-Lehmer primality test for Mersenne numbers
- implementation of the modular reduction discussed in the course (2p)
- the actual test (2p)

---
- comparisons between the version with the usual modular reduction and the dedicated modular reduction (1p)
