## Homework requirement
1. For an moderate-sized prime p (e.g., p is on 32 bits), generate α, a primitive root modulo p and then compute logarithm modulo p to the base α of β, for an arbitrary β, using Shanks or Pollard algorithm (5p)

2. Implement the Silver-Pohlig-Hellman algorithm for computing discrete logarithms modulo a large prime p (e.g., p is on 1024 bits). Assume that p - 1 has only small prime divisors and that its prime factorization is known in advance (5p)

## Grading scale:
### Shanks or Pollard algorithm
- own implementation of generate α calculation and compute logarithm modulo p to the base α of β and testing this component 

### Silver-Pohlig-Hellman algorithm
- implementation for computing discrete logarithms 

---
- comparisons between the version with the usual modular reduction and the dedicated modular reduction
